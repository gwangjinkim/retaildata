# RetailData

A high-performance library for fetching, preparing, and exploring retail benchmark datasets.

## Key Features

- **Multi-Provider Support**: Kaggle, Hugging Face, UCI, OpenML, and HTTP.
- **Fast Data Prep**: Automatic conversion to Parquet using Polars.
- **ML Ready**: Native integration with PyTorch, TensorFlow, and JAX.
- **Optimized**: Parallel downloads and Lazy Loading support.
- **Data Exploration**: Integrated CLI for schema validation and summary statistics.
